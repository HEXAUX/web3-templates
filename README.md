## 🚀 Quick Start

1. **Clone this repository**
```bash
git clone https://github.com/HEXAUX/web3-templates.git
cd web3-templates
```

2. **Choose a template**
```bash
cd template-1-terminal  # or any other template
```

3. **Open in browser**
Simply open `index.html` in your browser to view the template
